# Views

This directory contains the SwiftUI views that make up the visual interface of the Parallax application.

## Key Components
* **ContentView.swift** (in parent dir): The root container that handles the main sidebar navigation.
* **ChatsView.swift**: The main messaging view containing the list of recent chats.
* **MessageThreadView.swift**: The detail view for a specific chat conversation.
* **PeopleOrbitView.swift**: A dynamic, node-based visualization of contacts.
* **BubbleLayout.swift**: A custom SwiftUI Layout used to position elements in the `PeopleOrbitView`.
* **ContactDetailSidebar.swift**: A sidebar that reveals details about a specific contact.
* **KnowledgeBaseView.swift**: The UI for interacting with locally augmented knowledge / LLM features.
